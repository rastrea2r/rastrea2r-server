Quick Start
===========

rastrea2r-server is available on PyPI and can be installed with `pip <https://pip.pypa.io>`_.

.. code-block:: console

    $ pip install rastrea2r_server

After installing rastrea2r-server you can use it like any other Python module.

Here is a simple example:

    # Fill this section in with the common use-case.
